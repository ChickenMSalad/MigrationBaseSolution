# P4.20 Connector Execution Profiles + Runtime Policies

This set adds the first operator-facing execution profile workspace for connector runtime policies.

## Adds

- Admin API endpoints under `/api/operational/connectors/execution-profiles`
- Static profile catalog scaffold for concurrency, retry, throttle, and backoff policy review
- UI workspace under `apps/migration-admin-ui/src/features/executionProfiles`
- Validation endpoint for policy guardrail review

## Intent

Execution profiles are the bridge between connector configuration and distributed runtime behavior. Later sets can persist these profiles into SQL and bind them to migration projects, connector registrations, and run launch requests.
