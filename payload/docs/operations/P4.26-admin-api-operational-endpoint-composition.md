# P4.26 — Admin API Operational Endpoint Composition

This set consolidates operational endpoint registration into a single extension.

Before:

```csharp
app.MapOperationalAuditTrailEndpoints();
app.MapOperationalNotificationEndpoints();
app.MapOperationalSlaSloEndpoints();
...
```

After:

```csharp
app.MapMigrationOperationalEndpoints();
```

This keeps Program.cs manageable as the operational surface grows.
