# P4.32 — Operational SQL Database Bootstrap

This set adds local/dev bootstrap tooling for the operational SQL backbone.

## Preview

```powershell
./scripts/sql/P4.32-Initialize-OperationalSqlDatabase.ps1
```

## Apply

```powershell
./scripts/sql/P4.32-Initialize-OperationalSqlDatabase.ps1 -Apply
```

Defaults:

```text
Server:   (localdb)\MSSQLLocalDB
Database: MigrationBaseSolution_Operational
Scripts:  database/sql/operational
```

## Validate

```powershell
./scripts/sql/P4.32-Test-OperationalSqlDatabase.ps1
```

## Custom database

```powershell
./scripts/sql/P4.32-Initialize-OperationalSqlDatabase.ps1 `
  -ServerInstance ".\SQLEXPRESS" `
  -DatabaseName "MigrationBaseSolution_Operational_Dev" `
  -Apply

./scripts/sql/P4.32-Test-OperationalSqlDatabase.ps1 `
  -ServerInstance ".\SQLEXPRESS" `
  -DatabaseName "MigrationBaseSolution_Operational_Dev"
```

## Notes

This is intentionally local/dev bootstrap tooling. Cloud SQL deployment should continue to use the Azure deployment scaffold and a later migration/deployment path.
