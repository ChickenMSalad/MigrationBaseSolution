# Admin Web Features

This folder is the canonical feature-grouping root for `src/Admin/Migration.Admin.Web`.

Existing migrated Admin Web files may still live under `src/pages`, `src/api`, and `src/types` while consolidation proceeds. Future migration sets should move one feature family at a time into these folders after confirming current imports and routes from the repository.

`apps/migration-admin-ui` is feature-source/reference only and is not the deployable Admin UI.
