# P4.4 SQL Operational Work Item Queue

This set adds the SQL-backed work-item queue primitives needed before Service Bus dispatching and distributed workers.

## Adds

- `Migration.Application.Operational.WorkItems.IOperationalWorkItemQueue`
- SQL implementation in `src/Core/Migration.Infrastructure.Sql`
- Admin API endpoints under `/api/operational/sql-backbone/work-items`
- SQL schema/index supplement at `database/sql/operational/003_create_operational_work_item_queue.sql`

## Design Rules

- Additive only.
- No inline package versions.
- No broad solution/folder mutation.
- SQL remains the durable operational state source.
- Service Bus dispatching is not introduced here; this set creates the durable queue model Service Bus can later coordinate against.

## Install

```powershell
./patches/P4.4-Install-SqlOperationalWorkItemQueue.ps1 -WhatIf
./patches/P4.4-Install-SqlOperationalWorkItemQueue.ps1 -Apply
./patches/P4.4-Validate-SqlOperationalWorkItemQueue.ps1

dotnet restore
dotnet build
```
