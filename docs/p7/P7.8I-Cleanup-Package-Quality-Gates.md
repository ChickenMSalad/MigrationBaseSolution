# P7.8I Cleanup Package Quality Gates

## Purpose

P7.8I adds repo-local quality gates for cleanup/drop-in packages before they are committed or used for deployment.

This set exists because the P7.8 cleanup effort exposed several avoidable packaging problems:

- validators that matched their own guardrail text;
- PowerShell string interpolation bugs such as `$path:` inside double-quoted strings;
- validators that depended on brittle literal text instead of semantic checks;
- scripts that did not tolerate empty result collections;
- package links and instructions being trusted before the package was actually verified.

## Placement

Files are intentionally placed under existing repo conventions:

- `docs/p7` for P7 cleanup documentation;
- `tools/runtime` for reusable runtime validation tooling;
- `tools` for the set-level validator;
- `config-samples` for sample rule configuration.

No runtime host code is changed in this set.

## Guardrail model

The quality gate does not attempt to prove every script is perfect. It catches the failure modes that have already caused cleanup churn:

1. fragile PowerShell colon interpolation;
2. accidental scan of `bin` / `obj` outputs;
3. `$MyInvocation.ScriptName` bootstrap assumptions;
4. validators that mention stale runtime table defaults without making them explicit legacy references;
5. missing docs/config/tool files for a cleanup set;
6. missing ZIP/apply/validate instructions in cleanup docs.

## Usage

Run the set validator after applying this package:

```powershell
.\tools\validate-p7.8i-cleanup-package-quality-gates.ps1
```

Run the broader script quality gate any time before committing future cleanup packages:

```powershell
.\tools\runtime\Test-RuntimeScriptQuality.ps1 `
  -RepoRoot . `
  -FailOnIssue
```

To validate a specific cleanup set layout:

```powershell
.\tools\runtime\Test-RuntimeCleanupSet.ps1 `
  -RepoRoot . `
  -SetName P7.8I `
  -FailOnIssue
```

## Non-goals

This set does not remove legacy SQL objects, mutate app settings, or deploy workers. It is a packaging/review safety net.
