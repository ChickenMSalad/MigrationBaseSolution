# P7.10G Runtime Cleanup Closeout Gate

Adds a bounded closeout gate for the P7.10 runtime cleanup work.

The gate runs only the configured P7.10 validators, writes a markdown report, and does not mutate SQL, app settings, project files, or cloud resources.

## Run

```powershell
.\tools\runtime\Invoke-RuntimeP710CloseoutGate.ps1 `
  -ConfigurationPath .\config-samples\runtime-p710-closeout-gate.sample.json `
  -OutputPath .\artifacts\runtime-cleanup-closeout\p7.10-closeout-report.md
```
