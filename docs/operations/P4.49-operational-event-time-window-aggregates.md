# P4.49 — Operational Event Time Window + Aggregates

This set adds time-window filters and aggregate summaries over durable operational events.

Updated route:

```text
GET /api/operational/events/query
```

New parameters:

```text
fromUtc
toUtc
```

New route:

```text
GET /api/operational/events/query/summary
```

The summary returns:

- total event count
- counts by severity
- counts by category
- counts by event type
