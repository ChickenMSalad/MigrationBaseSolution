# P4.79 — Replay Composition Consolidation

This set consolidates replay service and endpoint registration.

## Service consolidation

Program.cs should now contain one replay service registration:

```csharp
builder.Services.AddExecutionReplayServices(builder.Configuration);
```

Instead of many individual replay registrations.

## Endpoint consolidation

Operational endpoint composition should now contain one replay endpoint registration:

```csharp
endpoints.MapExecutionReplayEndpoints();
```

Instead of many individual replay endpoint registrations.

## Why

This reduces:

- Program.cs growth
- duplicate registration mistakes
- fragile installer anchors
- noisy future diffs
- endpoint composition clutter

No runtime behavior changes are intended.
