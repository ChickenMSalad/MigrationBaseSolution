# Admin Web Audit Trail Client

This document records the P10.2R consolidation step.

The canonical deployable Admin UI is `src/Admin/Migration.Admin.Web`.

The app at `apps/migration-admin-ui` remains feature-source/reference only until its useful operational surfaces have been migrated into the canonical Admin Web.

P10.2R migrates the Audit Trail client/page surface into the canonical app without changing routes. The page can be wired into navigation in a later route-focused set.
