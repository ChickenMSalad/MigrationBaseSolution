# P4.44 — Operational Event Snapshot Recorder

This set adds a hosted background service that can periodically persist operational metrics snapshots into:

```text
dbo.MigrationOperationalEvents
```

The recorder is disabled by default.

## Configuration

```json
{
  "OperationalEventSnapshots": {
    "Enabled": false,
    "IntervalSeconds": 300,
    "StartupDelaySeconds": 30
  }
}
```

## Local use

1. Apply the operational event table from P4.42:

```powershell
./scripts/sql/P4.32-Initialize-OperationalSqlDatabase.ps1 -Apply
```

2. Add the configuration section to local Admin API settings.

3. Set:

```json
"Enabled": true
```

4. Run Admin API.

The service will begin recording `ScheduledOperationalMetricsSnapshot` events after the startup delay.
