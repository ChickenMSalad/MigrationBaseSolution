# P4.80 — Operational Admin Composition Consolidation

This set consolidates the broader operational admin service registrations.

Program.cs should now have:

```csharp
builder.Services.AddOperationalAdminServices(builder.Configuration);
```

Instead of scattered operational registrations such as:

```csharp
builder.Services.AddScoped<IOperationalEventStore, SqlOperationalEventStore>();
builder.Services.AddScoped<IExecutionPlanStore, SqlExecutionPlanStore>();
builder.Services.AddScoped<IExecutionWorkItemQueueStore, SqlExecutionWorkItemQueueStore>();
builder.Services.AddExecutionReplayServices(builder.Configuration);
```

Replay still remains its own sub-composition via:

```csharp
services.AddExecutionReplayServices(configuration);
```

inside `AddOperationalAdminServices`.

This keeps Program.cs much smaller and reduces fragile installer anchors.
