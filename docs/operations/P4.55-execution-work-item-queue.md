# P4.55 — Execution Work Item Queue

This set introduces durable execution work items.

New table:

```text
dbo.MigrationExecutionWorkItems
```

New routes:

```text
POST /api/operational/execution-work-items/expand
POST /api/operational/execution-work-items/lease
POST /api/operational/execution-work-items/complete
POST /api/operational/execution-work-items/fail
GET  /api/operational/execution-work-items/{executionSessionId}/summary
GET  /api/operational/execution-work-items/{executionSessionId}/recent
```

Work item states:

- pending
- leased
- running
- completed
- failed
- dead-lettered

This is the first runnable queue primitive for future migration workers.
