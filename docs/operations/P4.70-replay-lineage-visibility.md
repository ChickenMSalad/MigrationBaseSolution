# P4.70 — Replay Lineage Visibility

This set adds replay lineage visibility.

New route:

```text
GET /api/operational/execution-replay/{executionSessionId}/lineage
```

The lineage response includes:

- root execution session
- source execution session
- replay depth
- replay scope
- ancestors
- direct derived replay children

The UI adds a lineage panel to the selected execution session.
