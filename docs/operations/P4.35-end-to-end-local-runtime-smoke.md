# P4.35 — End-to-End Local Runtime Smoke Orchestrator

This set adds a single smoke script for validating the local P4 runtime chain.

## Prerequisites

1. Initialize SQL:

```powershell
./scripts/sql/P4.32-Initialize-OperationalSqlDatabase.ps1 -Apply
./scripts/sql/P4.32-Test-OperationalSqlDatabase.ps1
```

2. Start the Admin API:

```powershell
dotnet run --project src/Core/Migration.Admin.Api/Migration.Admin.Api.csproj --urls "https://localhost:55436"
```

3. Optional: start the UI:

```powershell
cd apps/migration-admin-ui
npm run dev -- --host localhost --port 5174
```

## Run the smoke

```powershell
./scripts/smoke/P4.35-Invoke-EndToEndLocalRuntimeSmoke.ps1 -AllowUntrustedCertificate
```

## Skip database validation

```powershell
./scripts/smoke/P4.35-Invoke-EndToEndLocalRuntimeSmoke.ps1 -AllowUntrustedCertificate -SkipDatabaseValidation
```

## Skip UI build

```powershell
./scripts/smoke/P4.35-Invoke-EndToEndLocalRuntimeSmoke.ps1 -AllowUntrustedCertificate -SkipUiBuild
```

## What this validates

- operational SQL database tables
- `/api/operational/sql/health`
- `/api/operational/command-center/summary`
- governance endpoint facades
- UI build readiness
