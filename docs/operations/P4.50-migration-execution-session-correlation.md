# P4.50 — Migration Execution Session Correlation

This set introduces durable execution sessions and correlation support for operational events.

New table:

```text
dbo.MigrationExecutionSessions
```

Updated table:

```text
dbo.MigrationOperationalEvents
```

New event columns:

```text
ExecutionSessionId
MigrationRunId
```

New routes:

```text
POST /api/operational/execution-sessions
GET  /api/operational/execution-sessions/recent
```

Updated snapshot route:

```text
POST /api/operational/events/snapshot
```

Optional body:

```json
{
  "executionSessionId": "00000000-0000-0000-0000-000000000000",
  "migrationRunId": "00000000-0000-0000-0000-000000000000"
}
```
