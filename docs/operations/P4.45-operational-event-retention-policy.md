# P4.45 — Operational Event Retention Policy

This set adds retention controls for durable operational events.

New routes:

```text
GET  /api/operational/events/retention/policy
POST /api/operational/events/retention/prune
```

The background retention worker is disabled by default.

Configuration:

```json
{
  "OperationalEventRetention": {
    "Enabled": false,
    "RetentionDays": 30,
    "IntervalHours": 24,
    "StartupDelaySeconds": 60
  }
}
```
