# P4.43 — Durable Audit Trail Event Reader

This set connects the audit-trail endpoint to the durable operational event store introduced by P4.42.

The audit recent endpoint now prefers:

```text
dbo.MigrationOperationalEvents
```

and only falls back to live SQL metric projection when no persisted events are available.

Route family:

```text
/api/operational/audit-trail
```

Useful manual flow:

```powershell
Invoke-RestMethod -Method Post -Uri "https://localhost:55436/api/operational/events/snapshot"
Invoke-RestMethod -Method Get  -Uri "https://localhost:55436/api/operational/audit-trail/recent"
```
