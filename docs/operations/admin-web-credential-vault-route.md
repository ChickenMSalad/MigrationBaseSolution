# Admin Web Credential Vault Route

Credential Vault is part of the canonical Admin Web application.

Canonical target:

```text
src/Admin/Migration.Admin.Web
```

Reference-only source:

```text
apps/migration-admin-ui
```

This route exposes the already migrated Credential Vault page at:

```text
/credential-vault
```

Do not create or deploy a second Admin UI from `apps/migration-admin-ui`.
