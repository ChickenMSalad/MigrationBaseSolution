# P4.52 — Execution Session Event Correlation UI

This set closes the first correlation loop between execution sessions and operational events.

Enhancements:

- event query accepts `executionSessionId`
- CSV export accepts `executionSessionId`
- execution-session UI can record a correlated snapshot
- execution-session UI can show session-specific events

This enables future runtime work to attach migration/connector execution history to a durable execution session.
