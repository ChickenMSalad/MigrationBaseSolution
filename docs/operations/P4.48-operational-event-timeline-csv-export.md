# P4.48 — Operational Event Timeline CSV Export

This set adds CSV export for filtered operational event timeline queries.

New route:

```text
GET /api/operational/events/export/csv
```

Supported query parameters:

- severity
- category
- eventType
- take

Example:

```powershell
Invoke-WebRequest `
  -Uri "https://localhost:55436/api/operational/events/export/csv?severity=critical&take=250" `
  -OutFile ".\operational-events.csv"
```

The UI timeline workspace now includes an **Export CSV** button using the current filters.
