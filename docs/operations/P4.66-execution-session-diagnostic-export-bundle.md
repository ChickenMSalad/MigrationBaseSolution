# P4.66 — Execution Session Diagnostic Export Bundle

This set adds a full execution-session diagnostic export.

New routes:

```text
GET /api/operational/execution-diagnostics/{executionSessionId}/bundle
GET /api/operational/execution-diagnostics/{executionSessionId}/bundle.json
```

The bundle includes:

- execution session
- queue summary
- execution plan steps
- work items
- phase history
- operational events correlated to the session
- worker telemetry summary

This is intended for audit/debug handoff and future replay/recovery workflows.
