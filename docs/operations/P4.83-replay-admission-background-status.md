# P4.83 — Replay Admission Background Status

This set exposes replay admission background automation status.

New route:

```text
GET /api/operational/execution-replay/admission/background/status
```

The response includes:

- whether the background evaluator is enabled
- evaluator interval
- evaluator take count
- whether admission governance is enabled
- max concurrent replay limit
- UTC admission window

The UI adds a status panel to the execution-session workspace.
