# P4.84 — Replay Admission Manual Controls

This set adds audited manual replay admission decisions.

New routes:

```text
POST /api/operational/execution-replay/admission/force-admit
POST /api/operational/execution-replay/admission/force-defer
```

Manual controls require:

- replay execution session id
- operator
- reason

Behavior:

- `force-admit` moves replay session to `queued`
- `force-defer` records a defer decision and leaves the replay session pending/queued as-is
- both write durable admission-decision records
- both write operational events
