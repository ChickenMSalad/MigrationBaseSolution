# P4.78 — Replay Admission Background Evaluator

This set adds an optional hosted service that periodically evaluates replay admission.

Configuration sample:

```text
config-samples/appsettings.ExecutionReplayAdmissionBackground.sample.json
```

Default behavior:

- disabled by default
- interval: 60 seconds
- take: 25 admission-pending replay sessions per pass

When enabled, the service calls the same replay admission logic introduced in P4.76.

This means replay sessions materialized as `admission-pending` can automatically move to `queued` when the configured admission rules allow it.
