# P4.42 — Durable Operational Event Snapshots

This set adds the first durable operational event table:

```text
dbo.MigrationOperationalEvents
```

New routes:

```text
POST /api/operational/events/snapshot
GET  /api/operational/events/recent
```

Apply SQL:

```powershell
./scripts/sql/P4.32-Initialize-OperationalSqlDatabase.ps1 -Apply
```

Record a snapshot through the Admin API:

```powershell
Invoke-RestMethod -Method Post -Uri "https://localhost:55436/api/operational/events/snapshot"
```
