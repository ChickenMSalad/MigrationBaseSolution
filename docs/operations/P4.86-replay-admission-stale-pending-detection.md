# P4.86 — Replay Admission Stale Pending Detection

This set adds replay admission health evaluation.

New route:

```text
POST /api/operational/execution-replay/admission/health/evaluate
```

Request:

```json
{
  "emitEvents": true,
  "take": 50
}
```

Configuration sample:

```text
config-samples/appsettings.ExecutionReplayAdmissionHealth.sample.json
```

The service detects replay sessions stuck in `admission-pending` beyond the configured threshold and can emit `ExecutionReplayAdmissionStalePending` warning events.
