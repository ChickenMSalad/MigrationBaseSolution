# P4.64 ‚Äî Execution Cancellation Controls

This set adds cancellation semantics for execution sessions.

New route:

```text
POST /api/operational/execution-control/cancel
```

Cancellation behavior:

- marks the execution session as `cancelled`
- sets `CompletedUtc`
- marks pending/leased/running/failed work items as `cancelled`
- clears worker/lease ownership
- writes `ExecutionSessionCancelled` operational event

Cancelled sessions already cannot lease new work because of the P4.62 lease guard.
