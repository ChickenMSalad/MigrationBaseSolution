# P4.60 — Execution Work Item Lease Renewal + Requeue

This set adds worker safety controls to the execution work-item queue.

New routes:

```text
POST /api/operational/execution-work-items/renew
POST /api/operational/execution-work-items/requeue
```

Renewal keeps a valid lease alive for long-running workers.

Requeue supports:

- failed items with retry capacity remaining
- expired leased items

The local worker harness now supports:

```powershell
-RenewLeaseBeforeCompletion
```
