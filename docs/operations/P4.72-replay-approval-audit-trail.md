# P4.72 — Replay Approval Audit Trail

This set adds replay approval history visibility.

New route:

```text
GET /api/operational/execution-replay/{sourceExecutionSessionId}/approvals?take=25
```

The endpoint returns approval records for a source execution session, including:

- approved
- consumed
- expired
- approval scope
- approver
- approval note
- replay execution session id when consumed

The UI adds a replay approval audit trail panel.
