# P4.69 — Replay Execution Session Materialization

This set creates a replay execution session from a prepared replay manifest.

New SQL migration:

```text
database/sql/operational/008_add_execution_replay_lineage.sql
```

New route:

```text
POST /api/operational/execution-replay/materialize
```

Behavior:

- requires approval note
- blocks unsafe replay preparation
- enforces replay depth limit
- creates a new queued replay execution session
- records source session lineage
- creates pending replay work items
- writes `ExecutionReplayMaterialized` event
