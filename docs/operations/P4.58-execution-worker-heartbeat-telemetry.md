# P4.58 — Execution Worker Heartbeat Telemetry

This set adds durable worker heartbeat telemetry.

New table:

```text
dbo.MigrationExecutionWorkerHeartbeats
```

New routes:

```text
POST /api/operational/execution-workers/heartbeat
GET  /api/operational/execution-workers/summary
```

The local P4.57 worker harness now emits heartbeats while it leases and processes execution work items.

This prepares the runtime for real distributed worker visibility.
