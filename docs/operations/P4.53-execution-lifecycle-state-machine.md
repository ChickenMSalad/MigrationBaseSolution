# P4.53 — Execution Lifecycle State Machine

This set introduces durable execution lifecycle tracking.

New table:

```text
dbo.MigrationExecutionPhaseHistory
```

New routes:

```text
GET  /api/operational/execution-lifecycle/phases
POST /api/operational/execution-lifecycle/transition
GET  /api/operational/execution-lifecycle/{executionSessionId}/history
```

Supported phases:

- created
- validating
- manifest-loading
- queued
- running
- paused
- failed
- completed
- cancelled

Each transition also writes a correlated operational event:

```text
ExecutionPhaseTransitioned
```

This is the first state-machine bridge toward real migration orchestration.
