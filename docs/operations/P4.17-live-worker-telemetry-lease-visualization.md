# P4.17 Live Worker Telemetry + Lease Visualization

P4.17 adds the first operator-facing worker telemetry workspace for the SQL-backed distributed runtime.

## Added areas

- `src/Core/Migration.Admin.Api/Endpoints/Operational/Workers/OperationalWorkerTelemetryEndpointExtensions.cs`
- `apps/migration-admin-ui/src/features/workers/*`

## Endpoint surface

- `GET /api/operational/workers/telemetry`
- `GET /api/operational/workers/leases`

The endpoint is intentionally a thin control-plane facade. It should be connected to the SQL work-item and lease projections as those contracts stabilize.
