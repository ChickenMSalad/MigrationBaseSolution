# P4.46 — Operational Event Timeline Query Surface

This set adds filtering and paging over durable operational events.

Route:

```text
GET /api/operational/events/query
```

Supported query parameters:

| Parameter | Description |
|---|---|
| severity | severity filter |
| category | category filter |
| eventType | event type filter |
| skip | paging offset |
| take | bounded page size |

Example:

```powershell
Invoke-RestMethod `
  -Method Get `
  -Uri "https://localhost:55436/api/operational/events/query?severity=critical&take=25"
```
