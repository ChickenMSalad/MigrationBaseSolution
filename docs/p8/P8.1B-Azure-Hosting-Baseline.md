# P8.1B Azure Hosting Baseline

This set documents the first cloud-hosting baseline for the SQL operational worker host.

## Scope

P8.1B is intentionally configuration-only. It does not change runtime code, package references, SQL schema, or worker behavior.

## Host

Primary host:

- `src\Hosts\Migration.Hosts.SqlOperationalWorker`

The host already supports:

- `appsettings.json`
- environment-specific appsettings
- local appsettings overlays
- user secrets
- `MIGRATION_` environment variables

## Required cloud settings

Minimum cloud configuration:

```json
{
  "ConnectionStrings:MigrationOperationalStore": "<sql connection string or Key Vault reference>",
  "SqlOperationalQueueExecutor:Enabled": "true",
  "SqlOperationalQueueExecutor:WorkerId": "sql-operational-worker-01",
  "SqlOperationalQueueExecutor:BatchSize": "10",
  "SqlOperationalQueueExecutor:LeaseSeconds": "300",
  "SqlOperationalQueueExecutor:PollDelaySeconds": "5",
  "SqlOperationalQueueExecutor:RunUntilIdleAndStop": "false",
  "SqlOperationalMigrationJobExecutor:Enabled": "false"
}
```

Do not configure `SqlOperationalQueueExecutor:RunId` except for short-lived debug runs.

## Azure App Settings equivalent

When using the `MIGRATION_` environment variable prefix, use double underscores for nested keys:

```text
MIGRATION_ConnectionStrings__MigrationOperationalStore
MIGRATION_SqlOperationalQueueExecutor__Enabled
MIGRATION_SqlOperationalQueueExecutor__WorkerId
MIGRATION_SqlOperationalQueueExecutor__BatchSize
MIGRATION_SqlOperationalQueueExecutor__LeaseSeconds
MIGRATION_SqlOperationalQueueExecutor__PollDelaySeconds
MIGRATION_SqlOperationalQueueExecutor__RunUntilIdleAndStop
MIGRATION_SqlOperationalMigrationJobExecutor__Enabled
```

## Initial production stance

Keep `SqlOperationalMigrationJobExecutor:Enabled=false` until connector/job execution cloud configuration is ready.

This allows the SQL operational runtime, readiness, discovery, and idle behavior to be validated before real migration jobs are allowed to execute.

## Validation

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\Validate-P81BAzureHostingBaseline.ps1
```
