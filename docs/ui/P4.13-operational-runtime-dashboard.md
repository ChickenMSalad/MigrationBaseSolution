# P4.13 Operational Runtime Dashboard

This set adds a build-safe React dashboard surface for the SQL-backed operational runtime.

## Added files

```text
apps/migration-admin-ui/src/lib/operationalRuntimeApi.ts
apps/migration-admin-ui/src/components/RuntimeStatusBadge.tsx
apps/migration-admin-ui/src/components/OperationalRuntimeDashboard.tsx
```

## Intended API contracts

The dashboard reads from the Admin API using these expected runtime endpoints:

```text
GET /api/operational/runtime/readiness
GET /api/operational/runs
GET /api/operational/work-items/summary
```

Each read has a safe fallback so the UI remains buildable while backend endpoint shapes continue to mature.

## Wiring

The installer attempts to wire the dashboard into `src/App.tsx` by adding:

```tsx
import { OperationalRuntimeDashboard } from './components/OperationalRuntimeDashboard';
```

and inserting:

```tsx
<OperationalRuntimeDashboard />
```

before the first `</main>` tag.

If `App.tsx` has been heavily changed, the installer copies the dashboard components and skips the app patch rather than corrupting the UI shell.
