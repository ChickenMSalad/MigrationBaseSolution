# P4.65 — Execution Cancellation Operator UI

This set exposes P4.64 cancellation controls in the execution-session UI.

Capabilities:

- cancel selected execution session
- reuse control reason as cancellation reason
- disable lease/queue/recovery actions for terminal sessions
- make cancelled status visible beside selected-session controls

Terminal statuses treated as locked in the UI:

- cancelled
- completed
- failed
