# P4.63 — Execution Pause Resume Operator UI

This set wires the P4.62 execution-control endpoints into the execution-session UI.

Capabilities:

- pause selected execution session
- resume selected execution session
- capture pause/resume reason
- disable UI lease action while selected session is paused
- show selected session status near controls

Backend behavior remains from P4.62: paused sessions do not lease new work.
