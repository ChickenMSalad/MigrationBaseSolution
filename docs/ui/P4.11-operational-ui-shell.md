# P4.11 Operational UI Shell

P4.11 introduces the first isolated frontend shell for the cloud control plane.

## Added path

```text
apps/migration-admin-ui
```

## Design constraints

- No .NET solution mutation
- No `Directory.Packages.props` changes
- No backend endpoint shape changes
- API base URL is environment-driven
- UI tolerates unavailable endpoints and displays failures instead of crashing

## Initial API probes

```text
/api/system/endpoints
/api/operational/sql-backbone/health
/api/operational/sql-backbone/summary
/api/operational/runtime-readiness/queue
/api/operational/runtime-readiness/runs
/api/operational/runtime-readiness/workers
```
