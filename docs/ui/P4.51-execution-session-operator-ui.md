# P4.51 — Execution Session Operator UI

This set adds a UI workspace for migration execution sessions.

The workspace uses:

```text
POST /api/operational/execution-sessions
GET  /api/operational/execution-sessions/recent
```

This is the first operator-facing bridge from the P4 operational control plane into actual migration execution workflow.

Operators can now create an execution session and copy its `executionSessionId` for event correlation.
