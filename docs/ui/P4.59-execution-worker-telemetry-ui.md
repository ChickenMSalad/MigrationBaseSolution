# P4.59 — Execution Worker Telemetry UI

This set adds a UI workspace for execution worker heartbeat telemetry.

The workspace reads:

```text
GET /api/operational/execution-workers/summary
```

It displays:

- total workers
- active workers
- idle workers
- stale workers
- worker id
- status
- active lease count
- execution session correlation
- last heartbeat

Run the P4.57 local worker harness after P4.58 to populate worker telemetry.
