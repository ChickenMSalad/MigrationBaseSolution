# P10.2BM - Admin Web Reference Quarantine Completion

## Purpose

After the Admin Web feature consolidation, copied `/apps` source can leave reference-only feature folders inside the compiled canonical `src/features` tree. This set quarantines ungrouped apps-style feature folders into the Admin Web reference area while preserving canonical grouped feature folders.

## Canonical folders preserved

- `connectors`
- `governance`
- `operations`
- `platform`
- `security`

## Behavior

The apply script:

1. Resolves the repo root from its script location.
2. Locates `src/Admin/Migration.Admin.Web/src/features`.
3. Checks known ungrouped apps-style residual folders.
4. Scans canonical Admin Web source files for references to each candidate folder.
5. Moves only unreferenced residual folders to `src/Admin/Migration.Admin.Web/reference/apps-migration-admin-ui/src/features`.
6. Writes a report to `docs/P10/P10.2BM-AdminWebReferenceQuarantineCompletion.Report.md`.

No source imports or routes are changed.
