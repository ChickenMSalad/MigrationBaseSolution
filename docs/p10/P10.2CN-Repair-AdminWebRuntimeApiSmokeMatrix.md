# P10.2CN Repair - Admin Web Runtime API Smoke Matrix

Repair-only package for the optional runtime API smoke matrix runner.

The runner now writes a summary immediately, uses per-request timeouts, limits endpoint probes, and always writes evidence files under `artifacts/p10/P10.2CN-Repair`.

Run from the repository root:

```powershell
powershell -ExecutionPolicy Bypass -File .\tools\p10\P10.2CN-Repair\Run-P10.2CN-Repair-AdminWebRuntimeApiSmokeMatrix.ps1
```
