# P10.2CL - Admin Web Endpoint Contract Inventory

## Purpose

The Admin Web now builds and has local smoke/deployment harnesses. This set adds an endpoint-contract inventory runner that reads the canonical Admin Web API source files and reports likely HTTP endpoint usage.

## Scope

- `src/Admin/Migration.Admin.Web/src/api/**/*.ts`
- `src/Admin/Migration.Admin.Web/src/features/**/api/**/*.ts`
- excludes `reference`, `node_modules`, `dist`, `bin`, and `obj`

## Output

- `artifacts/p10/P10.2CL/admin-web-endpoint-contract-inventory.md`
- `docs/P10/P10.2CL-AdminWebEndpointContractInventory.Report.md`

## Notes

This is report-only. It does not change application source, route definitions, Vite config, or package dependencies.
