# P10.2AO - Admin Web Connector Configuration Feature Move

## Scope

Moves only Connector Configuration into the canonical Admin Web feature folder:

- `src/Admin/Migration.Admin.Web/src/features/connectors/configuration/pages/ConnectorConfiguration.tsx`
- `src/Admin/Migration.Admin.Web/src/features/connectors/configuration/api/connectorConfigurationApi.ts`
- `src/Admin/Migration.Admin.Web/src/features/connectors/configuration/types/connectorConfiguration.ts`

## Non-goals

- No `/apps/migration-admin-ui` changes.
- No route reshaping beyond the import source in `App.tsx` when the route symbol is already used.
- No broad Admin Web feature migration.

## Safety notes

The scripts are intended to be PowerShell 5.1 and `Set-StrictMode -Version 2.0` safe. They validate move state before touching files, avoid nested validation arrays, avoid self-scanning validators, and avoid `$var:` string interpolation patterns.
