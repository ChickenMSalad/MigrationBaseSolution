# P10.2CD - Admin Web Admin API Connectivity Smoke

This package adds a local smoke runner for validating that the canonical Admin Web can target the Admin API through the configured Vite proxy target/base URL.

It is intentionally non-mutating for application source.

## Apply

```powershell
powershell -ExecutionPolicy Bypass -File .\tools\p10\P10.2CD\Apply-P10.2CD-AdminWebAdminApiConnectivitySmoke.ps1
powershell -ExecutionPolicy Bypass -File .\tools\p10\P10.2CD\Test-P10.2CD-AdminWebAdminApiConnectivitySmoke.ps1
```

## Optional local smoke

Start the Admin API separately, then run:

```powershell
powershell -ExecutionPolicy Bypass -File .\tools\p10\P10.2CD\Run-P10.2CD-AdminApiConnectivitySmoke.ps1
```

You may override the API URL:

```powershell
powershell -ExecutionPolicy Bypass -File .\tools\p10\P10.2CD\Run-P10.2CD-AdminApiConnectivitySmoke.ps1 -AdminApiBaseUrl https://localhost:55436
```

The runner writes artifacts under `artifacts/p10/P10.2CD/`.
