# P10.2BT - Admin Web Local Build Verification Harness

This set introduces a repeatable local Admin Web build verification harness for the canonical deployable Admin UI.

Scope:

- Validate `src/Admin/Migration.Admin.Web/package.json` exists.
- Validate `src/Admin/Migration.Admin.Web/tsconfig.json` exists.
- Validate `src/Admin/Migration.Admin.Web/vite.config.ts` exists.
- Validate the package has an npm `build` script.
- Generate a report under `docs/P10`.
- Add a dedicated local build runner for `npm run build`.

Non-goals:

- No source movement.
- No `App.tsx` rewrite.
- No route changes.
- No package version changes.
