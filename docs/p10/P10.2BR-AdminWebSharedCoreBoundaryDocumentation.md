# P10.2BR - Admin Web Shared Core Boundary Documentation

## Purpose

The canonical Admin Web consolidation has moved feature-specific page/API/type assets under `src/features/<domain>/<feature>`.

The remaining flat folders under `src/Admin/Migration.Admin.Web/src/api` and `src/Admin/Migration.Admin.Web/src/types` are now treated as shared/core surfaces unless a later targeted cleanup proves otherwise.

## Scope

This set adds documentation and validation only.

It does not:

- move feature files,
- touch `App.tsx`,
- rewrite imports,
- remove `/apps`,
- modify project/package configuration.

## Expected Outcome

After this set:

- `src/api/README.md` explains the shared/core API boundary.
- `src/types/README.md` explains the shared type boundary.
- `docs/P10/P10.2BR-AdminWebSharedCoreBoundaryDocumentation.Report.md` records the local counts and posture.
