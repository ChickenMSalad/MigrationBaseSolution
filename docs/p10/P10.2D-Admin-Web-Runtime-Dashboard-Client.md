# P10.2D - Admin Web Runtime Dashboard Client

## Purpose

Adds a React-side runtime dashboard client and page that consume the Admin API runtime dashboard endpoints already present in the repo:

- `GET /api/runtime/dashboard/summary`
- `GET /api/runtime/dashboard/runs`
- optional/detail endpoint from P10.2C when present: `GET /api/runtime/dashboard/runs/{runId}`

This set intentionally does not patch `App.tsx` or `Layout.tsx`. Routing/navigation is isolated for the next set so we do not guess against the current committed SPA layout.

## Files added

- `src/Admin/Migration.Admin.Web/src/types/runtimeDashboard.ts`
- `src/Admin/Migration.Admin.Web/src/api/runtimeDashboardApi.ts`
- `src/Admin/Migration.Admin.Web/src/pages/RuntimeDashboard.tsx`

## Expected build check

After applying, rebuild the Admin Web project:

```powershell
Push-Location .\src\Admin\Migration.Admin.Web
npm install
npm run build
Pop-Location
```
