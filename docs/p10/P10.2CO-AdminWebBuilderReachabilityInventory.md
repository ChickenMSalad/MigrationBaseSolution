# P10.2CO - Admin Web Builder Reachability Inventory

## Purpose

Verify and document whether Manifest Builder, Taxonomy Builder, and Mapping Builder survived consolidation as reachable Admin Web pages.

## Behavior

The apply script searches the local working tree for builder page candidates under:

- `src/Admin/Migration.Admin.Web/src/features`
- `src/Admin/Migration.Admin.Web/src/pages`
- `src/Admin/Migration.Admin.Web/reference/apps-migration-admin-ui/src/features`
- `src/Admin/Migration.Admin.Web/reference/apps-migration-admin-ui/src/pages`

For each builder:

- If a canonical page exists, the script ensures `App.tsx` has an import and route.
- If no canonical page exists but reference candidates exist, the script records restoration candidates for a later compile-safe restore.
- If no candidates exist, the script records that explicitly.

The set does not delete, overwrite, or copy builder files.
