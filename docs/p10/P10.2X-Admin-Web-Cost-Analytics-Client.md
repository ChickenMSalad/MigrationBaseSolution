# P10.2X - Admin Web Cost Analytics Client

## Purpose

P10.2X migrates the Cost Analytics client surface into the canonical Admin Web application at `src/Admin/Migration.Admin.Web`.

`apps/migration-admin-ui` remains feature-source/reference only. New deployable Admin UI work belongs in `src/Admin/Migration.Admin.Web`.

## Scope

This set adds:

- Cost analytics TypeScript contracts
- Cost analytics Admin API client
- Cost analytics page shell for operator visibility

This set intentionally does not patch routing/navigation. Route wiring should be handled in a separate bounded set after the current `App.tsx` and `Layout.tsx` shape is checked from the committed repository.
