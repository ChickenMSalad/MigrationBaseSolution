# P10.2CJ - Admin Web CI Build Validation Workflow

## Purpose

The canonical Admin Web now builds locally after consolidation. This set adds a GitHub Actions workflow that validates the Admin Web package independently whenever relevant files change.

## Scope

- Adds `.github/workflows/admin-web-build-validation.yml` when missing.
- Validates the workflow is path-scoped to `src/Admin/Migration.Admin.Web/**`.
- Validates the workflow runs `npm ci` and `npm run build` from the canonical Admin Web folder.

## Non-goals

- No Admin Web source rewrites.
- No deployment secrets.
- No cloud deployment job.
- No changes to existing runtime workflows.
