## Purpose

Continue P10.2 Admin UI consolidation by moving one remaining canonical Admin Web feature into the established `features` structure.

## Scope

- Cost Analytics

Target layout:

- `src/Admin/Migration.Admin.Web/src/features/platform/costAnalytics/pages/CostAnalytics.tsx`
- `src/Admin/Migration.Admin.Web/src/features/platform/costAnalytics/api/costAnalyticsApi.ts`

## Notes

The current canonical `App.tsx` may or may not import `CostAnalytics`. The apply script updates the import when it exists, treats an already-correct import as complete, and does not add an unused import when the page is not currently wired.

The apply script validates all required source files before moving anything when starting from the flat state. It is also idempotent for a completed state.
