# P10.2BO - Admin Web Navigation Reachability Audit

Report-only package for canonical Admin Web navigation/route coverage.

Purpose:
- inventory `App.tsx` route paths and route target names from the local working tree;
- inventory canonical feature pages under `src/Admin/Migration.Admin.Web/src/features`;
- identify feature pages that do not appear to have an `App.tsx` route;
- identify route target names that do not resolve to an existing feature page;
- note possible navigation files such as Layout/sidebar/nav components for follow-up review.

This package intentionally avoids source rewrites.
