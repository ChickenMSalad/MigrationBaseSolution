# P10.2P — Admin Web Execution Worker Telemetry Client

## Purpose

Migrate the execution-worker telemetry viewing capability from the feature-source Admin UI into the canonical Admin Web application.

## Canonical rule

- Canonical UI: `src/Admin/Migration.Admin.Web`
- Feature source/reference only: `apps/migration-admin-ui`

This set adds the client-side page, API wrapper, and types to canonical Admin Web. Route/navigation wiring is intentionally left to a separate set because `App.tsx` is currently compacted onto one line and previous automatic route patchers have proven fragile.

## Added files

- `src/Admin/Migration.Admin.Web/src/types/executionWorkerTelemetry.ts`
- `src/Admin/Migration.Admin.Web/src/api/executionWorkerTelemetryApi.ts`
- `src/Admin/Migration.Admin.Web/src/pages/ExecutionWorkerTelemetry.tsx`

## API contract

The client calls:

```text
GET /api/operational/execution-workers/summary?staleAfterSeconds={seconds}
```
