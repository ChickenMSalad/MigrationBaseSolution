# P10.3E Repair2 - Admin Web Operator Workflow Acceptance

Purpose: repair the operator workflow runtime runner so local HTTPS Admin API probes do not depend on a PowerShell certificate callback script block.

The runner uses curl.exe with -k and --max-time for API probes, which avoids the PS5.1 runspace issue caused by ServerCertificateValidationCallback script blocks.

No Admin Web or Admin API source files are modified by this set.
