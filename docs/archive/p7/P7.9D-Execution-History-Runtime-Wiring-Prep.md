# P7.9D Execution History Runtime Wiring Prep

This set prepares the runtime wiring step for SQL operational execution history without mutating the worker or DI registrations.

Why this is separate:

- P7.9B added the SQL execution-history writer surface.
- P7.9C inventoried the runtime wiring locations.
- P7.9D captures the exact current repo symbols before the runtime worker is modified.

This prevents another round of guessed method names or mismatched paths.

## Run order

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\Validate-P79DExecutionHistoryRuntimeWiringPrep.ps1
powershell -ExecutionPolicy Bypass -File .\scripts\Write-P79DExecutionHistoryRuntimeWiringPatchPlan.ps1
```

The second command writes:

```text
docs\p7\P7.9D-Execution-History-Runtime-Wiring-Patch-Plan.generated.md
```
