# P7.11A — Workspace Artifact Hygiene

## Purpose

P7.11A inventories generated, local-only, and runtime evidence artifacts that accumulated during P7.8–P7.10 stabilization. It does not delete files automatically.

## Why this exists

The runtime cleanup effort created useful evidence, but also many local artifacts that should not become part of the handoff path by accident. This set separates:

- durable repo documentation and scripts
- generated runtime evidence
- local publish outputs
- temporary smoke/parity outputs
- historical/debugging files

## Expected workflow

1. Run `New-RuntimeWorkspaceArtifactInventory.ps1`.
2. Review the generated markdown inventory.
3. Decide which artifacts should be deleted, archived, or kept.
4. Do not commit generated `artifacts/` output unless intentionally preserving evidence.

## No automatic deletion

This set intentionally avoids destructive cleanup. Any deletion should happen through a reviewed command list in a future set.
