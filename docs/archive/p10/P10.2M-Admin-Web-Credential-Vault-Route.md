# P10.2M - Admin Web Credential Vault Route

## Purpose

Wire the canonical Admin Web Credential Vault page into the solution-integrated Admin Web application.

The canonical Admin Web remains:

```text
src/Admin/Migration.Admin.Web
```

The feature-source app remains reference-only:

```text
apps/migration-admin-ui
```

## Scope

This set does not add another UI project. It only wires the existing canonical page:

```text
src/Admin/Migration.Admin.Web/src/pages/CredentialVault.tsx
```

into:

```text
src/Admin/Migration.Admin.Web/src/App.tsx
src/Admin/Migration.Admin.Web/src/components/Layout.tsx
```

## Expected route

```text
/credential-vault
```

## Validation

Run:

```powershell
.\tools\runtime\Apply-P102AdminWebCredentialVaultRoute.ps1
.\tools\validate-p10.2m-admin-web-credential-vault-route.ps1
```

Then rebuild Admin Web.
