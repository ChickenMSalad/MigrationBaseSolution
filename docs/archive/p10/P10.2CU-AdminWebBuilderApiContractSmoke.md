# P10.2CU - Admin Web Builder API Contract Smoke

Adds a bounded, non-mutating smoke runner for Manifest, Taxonomy, and Mapping builder API endpoint candidates discovered from the canonical Admin Web source tree.

The runner:
- accepts an Admin API base URL parameter;
- writes summary and CSV details immediately under artifacts/p10/P10.2CU;
- applies per-request timeouts;
- skips action-like endpoints instead of POSTing blindly;
- does not mutate Admin Web source files.

Example:

```powershell
powershell -ExecutionPolicy Bypass -File .\tools\p10\P10.2CU\Run-P10.2CU-AdminWebBuilderApiContractSmoke.ps1 -AdminApiBaseUrl "https://localhost:55436"
```
