# P10.2CQ - Admin Web Canonical Builder Route Restoration

Adds guarded route reachability for builder pages that already exist in the canonical Admin Web source tree.

Rules:

- Manifest Builder route: `/manifest-builder`
- Taxonomy Builder route: `/taxonomy-builder`
- Mapping Builder route: `/mapping-builder`
- No `.tsx` extension-bearing imports are generated.
- No reference-only source is promoted automatically.
- Ambiguous or missing builder candidates are reported only.
