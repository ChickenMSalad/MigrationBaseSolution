# P10.2T - Admin Web Consolidated Operations Routes

## Purpose

Expose the canonical Admin Web operational pages that were migrated from the feature-source app into `src/Admin/Migration.Admin.Web`.

The canonical deployable Admin UI remains:

```text
src/Admin/Migration.Admin.Web
```

The feature-source/reference app remains:

```text
apps/migration-admin-ui
```

This set does not add new pages. It wires existing canonical pages into the Admin Web router and navigation.

## Routes covered

- `/runtime-dashboard`
- `/runtime-runs/:runId`
- `/execution-sessions`
- `/failure-retry`
- `/credential-vault`
- `/connector-configuration`
- `/execution-worker-telemetry`
- `/notification-routing`
- `/audit-trail`
- `/command-center`

## Validation

Run:

```powershell
.\tools\runtime\Apply-P102AdminWebConsolidatedOperationsRoutes.ps1
.\tools\validate-p10.2t-admin-web-consolidated-operations-routes.ps1
```

Then rebuild the canonical Admin Web.
