# P10.2BJ - Admin Web Compile Scope Reference Hygiene

The canonical deployable Admin Web remains:

`src/Admin/Migration.Admin.Web/src`

This package explicitly excludes the reference/source-copy locations from the Admin Web TypeScript compile scope and verifies there are no source imports into the reference area.
