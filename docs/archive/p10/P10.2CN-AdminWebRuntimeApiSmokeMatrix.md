# P10.2CN - Admin Web Runtime API Smoke Matrix

Purpose: capture the Admin Web API surface that should be verified against a running Admin API during local site-up work.

This set is intentionally report/harness only:

- no Admin Web source rewrites
- no route rewrites
- no package changes
- no reference-folder imports

The optional runner creates an artifact under `artifacts/p10/P10.2CN` with:

- Admin Web source API files discovered locally
- static API paths discovered from those files
- safe GET-style probe results against a running Admin API
- skipped non-GET/write-oriented endpoints
