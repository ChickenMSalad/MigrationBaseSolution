# P10.2CF - Admin Web Deployment Readiness Verification

This set documents and verifies the canonical Admin Web deployment surface.

Canonical deployable UI:

```text
src/Admin/Migration.Admin.Web
```

Reference-only UI material:

```text
src/Admin/Migration.Admin.Web/reference/apps-migration-admin-ui
```

## What this set checks

- Admin Web package/build files exist.
- Vite config exists.
- TypeScript config exists.
- package-lock exists.
- reference material remains outside compiled source.
- a repeatable production build verifier exists under tools/p10/P10.2CF.

## What this set does not do

- It does not change React source.
- It does not change App.tsx.
- It does not modify package.json.
- It does not deploy to Azure.

## Next phase after this set

After this verification passes, the next P10 site-up work can focus on deployment/configuration wiring:

- static web app or app-service hosting posture
- Admin Web environment variable mapping
- Admin API base URL configuration
- cloud deployment smoke verification
