# P10.2AN Repair2 - Admin Web Credential Vault Feature Move

This repair completes the Credential Vault feature move after a partial P10.2AN/Repair apply.

It is intentionally state-based and idempotent:
- accepts files already moved into `src/Admin/Migration.Admin.Web/src/features/security/credentialVault`
- moves any remaining flat page/API/type files when present
- normalizes imports by matching actual import lines instead of stale exact text
- updates `App.tsx` only from the legacy Credential Vault page import to the feature import

powershell -ExecutionPolicy Bypass -File .\tools\p10\P10.2AN-Repair2\Apply-P10.2AN-Repair2-AdminWebCredentialVaultFeatureMove.ps1
