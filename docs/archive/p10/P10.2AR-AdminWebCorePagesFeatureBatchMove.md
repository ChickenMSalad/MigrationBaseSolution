# P10.2AR - Admin Web Core Pages Feature Batch Move

## Purpose

Continue Admin Web consolidation faster by batching remaining canonical flat page moves.

## Scope

Moves remaining files from:

- `src/Admin/Migration.Admin.Web/src/pages`

into canonical feature folders under:

- `src/Admin/Migration.Admin.Web/src/features`

The script moves only files that exist in the local checkout and skips already-moved targets.

## Safety

- PowerShell 5.1 compatible.
- `Set-StrictMode -Version 2.0` safe.
- Uses `[pscustomobject]` move specs, not nested arrays.
- Validates destination conflicts before moving.
- Does not recursively scan and flag itself.
- Uses `Join-Path` and relative import normalization; no corrupted escape sequences.
