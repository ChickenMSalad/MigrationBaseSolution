# P10.3A - Admin Web Runtime Acceptance Harness

Purpose: provide a bounded runtime acceptance probe for the canonical Admin Web and Admin API.

This set does not modify Admin Web or Admin API source files.

Expected manual prerequisites:

- Admin API is running locally.
- Admin Web is running locally via Vite dev server or preview server.
- Use the runtime runner with the local Admin Web and Admin API base URLs.

Runner:

```powershell
powershell -ExecutionPolicy Bypass -File .\tools\p10\P10.3A\Run-P10.3A-AdminWebRuntimeAcceptance.ps1 -AdminWebBaseUrl "http://localhost:5173" -AdminApiBaseUrl "https://localhost:55436"
```
