# P10.2W Admin Web Capacity Forecast Client

This set migrates the capacity forecast operator workspace into the canonical Admin Web application.

- Canonical UI: `src/Admin/Migration.Admin.Web`
- Feature-source only: `apps/migration-admin-ui`
- Adds a capacity forecast API client, types, and page.
- Does not make `apps/migration-admin-ui` deployable.
- Does not patch routes; route/nav exposure should be performed in a separate route wiring set after the current `App.tsx` and `Layout.tsx` shape is reviewed.
