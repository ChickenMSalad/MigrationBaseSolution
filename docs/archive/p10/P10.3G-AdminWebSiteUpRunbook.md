# P10.3G - Admin Web Site-Up Runbook

## Scope

This set records the current site-up path after Admin Web consolidation and runtime acceptance. It does not change Admin Web or Admin API source code.

## Canonical local endpoints

- Admin Web: `http://localhost:5173`
- Admin API: `https://localhost:55436`

## Start sequence

1. Start Admin API using the repo-native host command or IDE launch profile.
2. Start Admin Web from `src\Admin\Migration.Admin.Web` using `npm run dev`.
3. Keep both processes running while executing runtime acceptance scripts.

## Accepted P10.3 evidence gates

- Runtime acceptance: Admin Web and Admin API reachable.
- Route acceptance: current Admin Web route table reachable.
- Browser runtime health: route and asset probes succeed.
- Operator workflow acceptance: current route/API smoke checks succeed.
