# P10.3D - Admin Web Page API Interaction Coverage

Purpose: inventory current Admin Web API usage and optionally probe safe GET endpoints against a running Admin API.

This set does not modify Admin Web or Admin API source files.

Run after Admin API is running:

```powershell
powershell -ExecutionPolicy Bypass -File .\tools\p10\P10.3D\Run-P10.3D-AdminWebPageApiInteractionCoverage.ps1 -AdminApiBaseUrl "https://localhost:55436"
```

Outputs:

- `artifacts/p10/P10.3D/page-api-interaction-coverage.summary.md`
- `artifacts/p10/P10.3D/page-api-interaction-coverage.details.csv`
