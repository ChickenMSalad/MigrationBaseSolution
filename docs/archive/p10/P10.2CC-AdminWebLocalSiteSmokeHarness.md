# P10.2CC - Admin Web Local Site Smoke Harness

## Purpose

Admin Web consolidation now builds. This set adds a repeatable local site-up smoke harness before moving into deeper Admin API/runtime integration work.

## Scope

- Validate canonical Admin Web package and Vite config files exist.
- Document local Admin Web runtime configuration.
- Add a local smoke runner for Vite dev server.
- Write artifacts under `artifacts/p10/P10.2CC`.

## Non-goals

- No Admin Web source rewrites.
- No route changes.
- No API implementation changes.
- No deployment changes.
