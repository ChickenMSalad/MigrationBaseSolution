# P10.3D Repair3 - Admin Web Page API Interaction Coverage

Repairs the page API coverage runner so it only probes literal HTTP API paths that start with `/api/`.

It skips relative module imports such as `../../../../api/client`, dynamic/template paths that cannot be safely parameterized, and non-GET interactions.

Run:

```powershell
powershell -ExecutionPolicy Bypass -File .\tools\p10\P10.3D-Repair3\Run-P10.3D-Repair3-AdminWebPageApiInteractionCoverage.ps1 -AdminApiBaseUrl "https://localhost:55436"
```
