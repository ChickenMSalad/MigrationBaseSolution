# P10.2CG - Admin Web Route Smoke Harness

## Purpose

The Admin Web now builds. This set adds a local route smoke harness so site-up work can verify the browser-facing SPA routes generated from the canonical `App.tsx` route table.

## Scope

- Adds a route smoke runner under `tools/p10/P10.2CG`.
- Adds a validation script for the runner and generated reports.
- Reads local `src/Admin/Migration.Admin.Web/src/App.tsx`.
- Extracts literal `path="..."` route entries.
- Sends HTTP GET requests to a running Admin Web dev server.
- Writes a Markdown report under `artifacts/p10/P10.2CG`.

## Non-goals

- No application source rewrites.
- No route additions/removals.
- No package changes.
- No dependency installation.
- No Admin API mutation.

## Expected usage

1. Start Admin Web locally.
2. Run the route smoke script from repo root.
3. Review the report for failed routes.
