# P10.2CE - Admin Web Local Stack Orchestration

Purpose: provide repeatable local scripts for starting the canonical Admin API and canonical Admin Web after P10.2 consolidation compile readiness.

Canonical Admin Web:

```text
src/Admin/Migration.Admin.Web
```

Canonical Admin API:

```text
src/Core/Migration.Admin.Api
```

Default local URLs:

```text
Admin API: https://localhost:55436
Admin Web: http://127.0.0.1:5173
```

Run the apply/test scripts from the repository root. The run scripts write logs under:

```text
artifacts/p10/P10.2CE
```

This package does not rewrite application source files.
