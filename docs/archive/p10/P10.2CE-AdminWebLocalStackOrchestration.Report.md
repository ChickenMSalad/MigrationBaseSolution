## Helper scripts

- `Run-P10.2CE-AdminApi.ps1` starts the Admin API with a configurable `--urls` value.
- `Run-P10.2CE-AdminWeb.ps1` starts the Admin Web Vite dev server with `VITE_ADMIN_API_PROXY_TARGET`.
- `Run-P10.2CE-LocalStackSmoke.ps1` checks the configured Admin API health URL and Admin Web URL.
