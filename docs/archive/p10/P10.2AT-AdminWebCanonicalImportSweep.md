# P10.2AT - Admin Web Canonical Import Sweep

This package is a pathing accelerator after the Admin Web feature/component batch copy and feature-folder moves.

It intentionally operates against the local working tree because failed repairs are not committed unless the user says they passed and committed.

Outputs:

- `docs/P10/P10.2AT-AdminWebCanonicalImportSweep.Report.md`

Rules:

- No `/apps` writes.
- No route patching.
- No file moves.
- Only relative import source normalization when the target import is currently unresolved and exactly one canonical target can be found.
