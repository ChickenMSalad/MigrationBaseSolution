# P10.2BU - Admin Web npm Dependency Restore Hygiene

## Purpose

The Admin Web build currently fails before TypeScript compilation because the local `node_modules` tree cannot resolve `node_modules/typescript/bin/tsc`.

This set adds local runners that:

1. remove stale TypeScript npm artifacts,
2. run a dev-inclusive npm install,
3. verify the TypeScript compiler entry point exists,
4. run the Admin Web build with stdout/stderr captured to artifacts.

## Scope

No Admin Web source files are moved or rewritten.
