# P10.2CY - Admin Web Builder Backend Verb-Aware Smoke

Adds a bounded, verb-aware smoke runner for restored builder backend contracts.

## Scope

- Manifest Builder build/validate/preview endpoints are probed with POST.
- Taxonomy Builder collection/validate/preview endpoints are probed with safe GET/POST semantics.
- Mapping Builder collection/profiles/validate/preview endpoints are probed with safe GET/POST semantics.
- Every request has a timeout and the runner always writes summary/details evidence.

## Optional runner

```powershell
powershell -ExecutionPolicy Bypass -File .\tools\p10\P10.2CY\Run-P10.2CY-AdminWebBuilderBackendVerbAwareSmoke.ps1 -AdminApiBaseUrl "https://localhost:55436"
```
