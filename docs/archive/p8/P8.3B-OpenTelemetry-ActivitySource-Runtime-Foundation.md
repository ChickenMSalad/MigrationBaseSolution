# P8.3B OpenTelemetry ActivitySource Runtime Foundation

This set adds the repo-native ActivitySource and tag-name baseline for future OpenTelemetry/Azure Monitor runtime wiring.

## Scope

Added:

- `OperationalExecutionActivitySources`
- `OperationalExecutionActivityTags`
- validation script
- inventory script

Not added:

- package references
- Azure Monitor exporter registration
- OpenTelemetry runtime registration
- worker runtime mutation
- SQL changes

## ActivitySource

Canonical source:

```text
Migration.Operational.Execution
```

Initial activity names:

- `migration.operational.sql_queue.work_item.execute`
- `migration.operational.servicebus.work_item.execute`
- `migration.operational.servicebus.dispatch`
- `migration.operational.run.evaluate`

## Tag policy

Activity tags should use stable lower-case dotted names such as:

- `migration.run.id`
- `migration.work_item.id`
- `migration.manifest_row.id`
- `messaging.servicebus.correlation_id`
- `messaging.message.id`
- `migration.execution.duration_ms`

## Next set

P8.3C should wire Activity creation around SQL queue and Service Bus execution boundaries without adding exporters yet.
