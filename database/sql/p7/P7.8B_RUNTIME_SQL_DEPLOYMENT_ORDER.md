# P7.8B Runtime SQL Deployment Order

Use this order for the active SQL operational runtime baseline:

1. `database/sql/p7/006_sql_operational_runtime_bootstrap_compatibility.sql`
2. `database/sql/p7/007_sql_operational_execution_history.sql`
3. `database/sql/p7/009_runtime_sql_contract_validator.sql`

Do not use these legacy GUID-era scripts for new Service Bus dispatcher/executor deployments:

- `database/sql/operational/001_create_operational_backbone.sql`
- `database/sql/operational/003_create_operational_work_item_queue.sql`
- `database/sql/p7/004_smoke_seed_operational_run.sql`
- `database/sql/p7/005_sql_operational_schema_compatibility_repair.sql` unless explicitly doing compatibility repair work

The validator is read-only and should be safe to run repeatedly against local and Azure databases.
